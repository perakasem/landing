## Stack

### Frontend

- **SvelteKit 2**
- **Svelte 5**
- **TypeScript**
- **Tailwind CSS 4**
- **Vite 6**

### Backend

- **Supabase**
- **Directus**

### Content

- **remark-gfm**
- **PostgreSQL**

---

## Performance

- **Site Config:** 5-minute server cache
- **Posts:** Always fresh from database
- **Static Assets:** CDN cached via Vercel edge
- **SSR:** Every page request
- **Code Splitting:** Automatic per route
